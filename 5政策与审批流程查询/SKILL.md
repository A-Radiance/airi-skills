# 建筑报建与审批政策查询技能

当用户咨询报建流程、政策依据或审查要点时，请按以下模块化逻辑执行：

### 1. 预处理：地域与时效定位
由于中国建筑政策具有强烈的“属地化”特征，在执行搜索前必须确认：
* **目标城市**：(如：上海、深圳、成都等)。
* **项目性质**：(如：民用建筑、工业建筑、改建项目)。
* **政策版本**：强制搜索近两年发布的最新“深化营商环境改革”后的流程。

### 2. 五大专项审查执行标准

#### A. 方案报建 (Planning Approval)
* **查询重点**：土地出让合同要求、容积率计算规则、建筑间距与退让、城市设计导则、建设工程规划许可证 (FGS) 办理要件。
* **核查清单**：总平面图、日照分析报告、交通影响评价。

#### B. 人防报建 (Civil Air Defense)
* **查询重点**：应建人防面积比例（按地上建筑面积百分比）、防护等级要求、易地建设费缴纳标准（如符合免建条件）。
* **核查清单**：人防方案设计、防护功能平战转换方案。

#### C. 消防报建 (Fire Safety)
* **查询重点**：特殊建设工程 vs 一般建设工程分类、消防设计审查制度、建筑高度/层数限制、疏散距离、耐火等级要求。
* **核查清单**：消防专项设计图纸、装修材料防火性能证明。

#### D. 施工图审查 (Construction Drawing Review)
* **查询重点**：数字化联合审图流程、结构安全专项、绿色建筑等级要求、节能计算表、海绵城市设计指标。
* **核查清单**：全套施工图、强条检查报告、工程勘察报告。

#### E. 超限审查 (Over-limit Structure Review)
* **查询重点**：超限高层建筑工程界定标准、抗震设防专项审查流程、专家评审会组织规程。
* **核查清单**：抗震加强措施说明、模型计算对比数据、专家评审费标准。

### 3. 输出详述模板
请按以下结构化方式为用户提供详述（禁止模糊概括）：

---
## [XX市] [审批专项名称] 政策详情简报

### ⚖️ 政策依据
* **核心法规**：列出具体的《XX市建设工程审批管理规定》及文号。
* **技术标准**：如《建筑设计防火规范》GB 50016 (最新版)。

### 📋 办理信息
| 维度 | 详细说明 |
| :--- | :--- |
| **申报主体** | 建设单位 / 设计单位配合 |
| **所需材料** | 1. ... 2. ... (需列出具体图纸及公章要求) |
| **审批时限** | 承诺办结天数（区分法定时限与承诺时限） |
| **办理入口** | XX市工程建设一体化平台 URL |
| **费用标准** | 规费、评审费或免征政策 |

### 🔍 审查核心要点 (Technical Focus)
* 列出该项审批中最容易被“弹回”或“不予通过”的技术难点（如日照不足、结构不规则等）。

### 📅 最新动向 (今年趋势)
* 提示当前是否推行“多审合一”、“告知承诺制”或“全流程网办”。
---

### 4. 异常处理
若查询不到特定城市的最新细则，应调用 `web_search` 访问该市“住房和城乡建设局”或“规划和自然资源局”官网进行实时抓取。
